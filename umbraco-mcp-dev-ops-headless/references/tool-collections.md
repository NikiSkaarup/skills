# Headless Dev Ops Tool Collections

Use the minimal set of collections needed for the task.

## Common Tasks

- Document Types and compositions -> `document-type`
- Data Types and property editor settings -> `data-type`, `property-type`
- Models Builder generation -> `models-builder`
- Index management and search -> `indexer`, `searcher`
- Webhooks for content events -> `webhook`
- Health checks and diagnostics -> `health`, `server`
- Log analysis -> `log-viewer`
- Permission checks -> `user`
- Preview templates (optional) -> `template`, `partial-view`

## Example Pairings

- Add a headless Document Type -> `document-type`
- Update content picker config -> `data-type`
- Rebuild a delivery index -> `indexer`
- Validate searcher health -> `searcher`
- Create a publish webhook -> `webhook`
